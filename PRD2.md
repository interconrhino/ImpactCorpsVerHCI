# PRD2: Our Program Page

## Goal
Create a new "Our Program" page that reuses the home page visual language (nav/header, hero layout, section rhythm, typography, CTA style) while using entirely new, student-first content.

## Audience
Prospective students exploring ImpactCorps and deciding whether to apply.

## Page Structure (Match Home Page Visual Language)
1) Header/nav (same styling as Home)
2) Hero section with CTA (same layout style as Home hero)
3) Section blocks with rounded panels and cards (same visual rhythm as Home)
4) CTA banner (same styling as Home)
5) Footer or closing section (same styling as Home)

## Section 1: Sign-Up Hero
### Purpose
Immediate student-oriented value proposition with a clear CTA.

### Slogan
"Build CV. Make an Impact with AI. Get paid."

### Copy Notes
- Short supporting line: Emphasize real projects, mentorship, and paid impact.
- Primary CTA: "Apply now"
- Secondary CTA: "See how it works"

## Section 2: Key Program Features
### Purpose
Detail what students actually get from the program.

### Feature List (draft)
- Community-rooted projects: Solve real issues defined by lived experience.
- AI tool fluency: Practical, hands-on use of AI for research, prototyping, and delivery.
- Mentorship + peer cohort: Feedback loops, weekly check-ins, and peer support.
- Impact-based pay: Rewards tied to adoption and measurable outcomes.
- Career outcomes: Portfolio-ready deliverables and CV-ready experience.

### Layout
3-column cards on desktop; stacked cards on mobile, same as Home feature grids.

## Section 3: Step-by-Step Student Journey
### Purpose
Make the program flow clear and predictable.

### Steps (draft)
1) Apply and get accepted
2) Onboarding + goal setting
3) Choose a community problem
4) Build with AI (guided sprints)
5) Launch + measure impact
6) Earn rewards + showcase your work

### Layout
Numbered list or card row with step labels.

## CTA Banner
Re-use the Home CTA panel style with updated copy:
- "Ready to build your impact portfolio?"
- CTA button: "Apply now"

## Success Criteria
- The page uses the same visual language, spacing, typography, and CTA styles as the Home page.
- The first three sections appear in the exact order defined above.
- Content is unique to this page and stays student-centered, concrete, and action-oriented.
